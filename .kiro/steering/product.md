# Nikke DB - Product Overview

## What is Nikke DB?

Nikke DB is a comprehensive database and utility application for the game **Nikke: Goddess of Victory**. It's a Vue 3-based web application that provides players with tools to explore, organize, and analyze game content.

## Core Features

- **Live2D Viewer**: Interactive character visualization using Live2D technology
- **Chibi Viewer**: Sprite-based character display with animations
- **Gallery**: Browse and view game artwork and promotional content
- **Tier List Maker**: Create and customize tier lists for characters, with export functionality
- **Tools**: Various utility tools for game data analysis and management
- **Credits**: Contributor and attribution information

## Target Users

- Nikke game players seeking character information and tools
- Content creators building tier lists and guides
- Community members exploring game assets

## Key Characteristics

- **Community-driven**: Open source (WTFPL license), built by and for the community
- **Asset-rich**: Hosts extensive game assets (character images, animations, gallery content)
- **Interactive**: Real-time visualization and manipulation of game data
- **Mobile-responsive**: Supports both desktop and mobile experiences
- **Data-persistent**: Uses local storage for saving user-created content (tier lists, preferences)

## Local Development

This is a local development version of Nikke DB. All assets and code are stored locally without external dependencies.
